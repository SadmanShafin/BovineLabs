# BovineLabs Quill Grid

BovineLabs Quill Grid adds grid-focused Quill components for DOTS projects built on top of BovineLabs Quill.

## Package name

`com.bovinelabs.quill.grid`

## Dependencies

This package depends on:
- `com.bovinelabs.core`
- `com.bovinelabs.quill`
- `com.unity.burst`
- `com.unity.collections`
- `com.unity.entities`
- `com.unity.mathematics`

See `package.json` for exact versions.

## Development

This package is intended to be tested from a host Unity project by embedding or referencing it as a package dependency.
